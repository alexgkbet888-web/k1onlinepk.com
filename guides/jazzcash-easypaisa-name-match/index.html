Name match matters

Wallet name mismatch can create delays even when the transaction proof looks correct. Check spelling, account ownership, and screenshots before assuming a withdrawal failed.

Proof before support

A useful proof set includes transaction ID, time, wallet method, account name, and the in-app status screen.

Do not retry blindly

Repeated deposits or withdrawals without checking the first status can make support review harder.

Mobile checklist

FAQ

Can name mismatch delay a withdrawal?

Yes. A mismatch can require review because the wallet account and game account may not appear connected.

What proof is useful for JazzCash or Easypaisa issues?

Use transaction ID, date, time, wallet number masking, and the in-app wallet status screen.

Should I make another transaction if one is pending?

Wait and document the first transaction before repeating, especially if support evidence may be needed.
